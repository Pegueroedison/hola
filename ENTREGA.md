# Entrega correcta

Esta entrega es la versión buena y rápida:

- React + Vite compilado para producción.
- Assets minificados y con caché por hash.
- Rutas relativas para funcionar en GitHub Pages, Netlify, Vercel, dominio propio o subcarpeta.
- Sin demo local.
- Sin HTML improvisado.
- Sin depender de `npm install` para ver la interfaz publicada.
- Protección para no quedar en pantalla blanca si Supabase todavía no está configurado.
- Código fuente completo incluido aparte.

## Qué subir

Sube todo lo que está dentro de esta carpeta.

El repositorio debe verse así:

```txt
index.html
assets/
icons/
manifest.webmanifest
404.html
.nojekyll
README.md
docs/
codigo-fuente-completo/
```

## Qué NO hacer

No subas el ZIP cerrado.
No subas solo `codigo-fuente-completo/`.
No dejes el `index.html` dentro de una carpeta adicional.
