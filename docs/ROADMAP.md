# Roadmap recomendado

## Fase 1: Núcleo

- Empresas, sucursales, usuarios, roles y permisos.
- Productos, inventario base y clientes.
- POS con ventas, pagos y caja.
- Offline queue básica.

## Fase 2: Operación

- Compras, proveedores, cuentas por cobrar y pagar.
- Kardex completo.
- Tickets 58/80mm y etiquetas.
- Autorizaciones completas.
- Motor de áreas operativas.

## Fase 3: Administración avanzada

- Contabilidad automática.
- Nómina y RRHH.
- Auditoría avanzada.
- Reportes PDF, Excel y CSV.
- Editor profesional de imágenes.

## Fase 4: Producción

- Firma real de Cloudflare R2.
- Integración de impresoras locales.
- Pruebas de carga.
- Pruebas de seguridad RLS.
- App Desktop firmada para Windows.
